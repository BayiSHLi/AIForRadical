# WordCloud 可视化 vs 真实频率分析指南

## 📊 **问题现象**

你看到的词云图中：
- **"one"** 和 **"muslim"** 看起来很大
- 但在 Top 30 统计中：
  - "jinny" = #1 (1190 次) - 词云中不太明显
  - "one" = #4 (767 次) - 词云中很大
  - "muslims" = #19 (422 次) - 词云中很大

**结论**: 词云的视觉大小 ≠ 实际频率！

---

## 🔧 **根本原因分析**

### **1. WordCloud算法问题**
```python
WordCloud不是线性映射词频 → 字体大小

实际映射逻辑：
词频 → 对数变换 → 字体大小计算 → 布局优化 → 最终视觉
      (非线性)           (复杂)      (冲突)
```

### **2. 字体尺寸限制**
```
理想情况:   freq=1190  →  size=100px
           freq=767   →  size=70px

实际情况:   频率高的词可能触碰字体上限 (比如 120px)
           无法继续放大，所以看不出区别
```

### **3. 视觉感受偏差**
```
心理因素:
- "one" (3字符)  看起来比 "jinny" (5字符) 更占据空间
- 某些颜色（黄色、绿色）看起来比蓝色更突出
- 位置：中心位置显得更重要
```

### **4. 布局优化冲突**
```
WordCloud目标：
1. 频率高 → 字体大 ✓
2. 美观展示 → 填充2D空间 ✗

结果：优先考虑美观，高频词可能被压小以容纳更多词汇
```

### **5. max_words 参数影响**
```python
# 原配置：max_words=200
# 问题：200个词 → 很多低频词也被显示
#      整体分布散开，高频词的相对优势不明显

# 改进：max_words=100
# 效果：只显示最高频100个词，使频率差异更明显
```

## 💡 **理论背景**

### **WordCloud 的数学模型**

```
频率 f(w)  → [对数变换] → [缩放] → 字体大小
           log(f)      * k    + b

具体公式：
font_size = log(frequency) * scaling + min_font_size

限制条件：
  min_font_size ≤ font_size ≤ max_font_size
  (比如: 10px ≤ size ≤ 120px)
```

**问题**: 高频词都会达到 max_font_size (120px)，无法区分

### **解决方案: relative_scaling 参数**

```python
# 原生 WordCloud
relative_scaling = 1.0  # 高频词完全支配大小

# 改进版本
relative_scaling = 0.5  # 降低频率影响，增加多样性

# 公式调整
font_size = log(frequency) ** relative_scaling * k + b
           # relative_scaling越小，曲线越平缓，字体大小差异越小
```

## 🔧 **参数调优指南**

### **如何让词云更接近真实频率**

```python
# 改进的WordCloud参数
kwargs = {
    'relative_scaling': 0.5,    # ← 关键：降低频率影响
    'max_words': 100,           # ← 降低显示词数，提高顶部词汇对比
    'min_font_size': 12,        # ← 抬高最小字体
    'width': 1400,              # ← 增大画布
    'height': 700,              # ← 增大画布
    'collocations': False,      # ← 禁用词组，只看单词
}
```

### **参数详解**

| 参数 | 默认值 | 改进值 | 说明 |
|------|--------|--------|------|
| `relative_scaling` | 1.0 | 0.5 | 越小 = 频率影响越小 = 字体大小差异越小 |
| `max_words` | 200 | 100 | 显示的最多词数 |
| `min_font_size` | 4 | 12 | 最小字体大小 |
| `prefer_horizontal` | 0.7 | 0.9 | 偏好水平排列，增加可读性 |
